Ready to Run the Full Application?

This is a preview of the Carbon Emissions Tracker. To run the full interactive application:

  1. Install Node.js from nodejs.org
  2. Open Command Prompt in this folder
  3. Run: npm install
  4. Run: cd server && npm install && cd ..
  5. Double-click start.bat or run servers manually

See SETUP.md for detailed instructions.
